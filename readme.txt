== Description ==

This plugin is used to find latitude and longitude of an address that you specify.
You can also drag the marker for more precision if the plugin result is not so good...
Then you can use the coordinates to create a new google map, to set a new poi in your 
car navigator or many others things that you can think!

History:
1.0.1 Add intenationalization
1.0.0 First release


== Installation ==

1. Upload the folder `paul_google_maps_coordinates` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[paul_google_maps_coordinates]` in your post or page
